# decision-tree

### Local setup

#### Dependencies

* [Elm `0.19`](https://guide.elm-lang.org/install.html)
* [`create-elm-app`](https://github.com/halfzebra/create-elm-app)

To run locally:

```bash
git clone git@github.com:kfrn/decision-tree.git
cd decision-tree/
npm install
npm run build-css
elm-app start
```

### Tests

```bash
elm-app test
```
